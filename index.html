HOW THIS STAYS OFFLINE

  1. Nothing is uploaded. The folder is read with your browser's local File System APIs — its bytes never go over the network.
  2. The ISO 9660 image is assembled in JavaScript, right here in the tab, sector by sector.
  3. Large files stream to disk in 4MB chunks via the File System Access API, so multi-gigabyte folders don't need to fit in memory.
  4. The app itself is cached by a service worker after your first visit, so you can reload this page with no connection at all.
